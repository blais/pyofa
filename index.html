Description

pyofa is a simple Python wrapper around the MusicDNS technology provided by MusicIP for generating PUIDs (audio fingerprints) from audio files (e.g. MP3, M4A, etc.).

Dependencies

  • Python 2.5 or higher
  • MusicIP's libofa
